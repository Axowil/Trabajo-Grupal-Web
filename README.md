## Descripción del proyecto
Reservas de Canchas Web es una aplicación web desarrollada como trabajo grupal para el curso de Introducción al Desarrollo Web, cuyo objetivo es permitir a los usuarios visualizar canchas deportivas disponibles y realizar reservas de manera sencilla mediante una interfaz web moderna e intuitiva. El proyecto está desarrollado con tecnologías de frontend y backend, junto a base de datos.

## Lenguajes y tecnologías utilizadas
- HTML5  
- CSS3  
- JavaScript (JS)  
- Python 
- SQL
